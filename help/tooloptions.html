Tool Options

This widget allows modification of the following :

Nothing is selected
nothing
Only wires are selected
nothing
At least one I/O or circuit is selected
A checkbox is provided to allow showing the component's name in the main view.
At least one circuit is selected
A checkbox is provided to allow showing the component's category (ie. And Gate, DMux, ....) in the main view.
Exactly one circuit or I/O is selected
A text field is provided to allow changing that component's name.
Only basic gates (and no NOT gate) are selected
A dropdown menu is provided to allow changing their number of inputs.
