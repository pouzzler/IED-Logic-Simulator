IED Logic Simulator User Manual

This user manual describes the various elements of the IED Logic Simulator's (IEDLS) user interface, and their interaction.

The IELDS is a logic circuit simulator with which the user can build and simulate gate-based logic circuits.

The construction is done by dragging basic gates and I/O from the toolbox widget to the main area of the program's interface, optionally modifying some of their characteristics using the tool options widget or the contextual right-click menu and connecting them together using the mouse in the main area.

The simulation consists in setting the circuit's inputs (for example with the contextual menu), and in watching the circuit's internals and outputs evolve according to the rules of boolean logic.
