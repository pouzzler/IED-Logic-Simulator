Menu

The menus are straightforward and don't need much explanation:

The file menu allows saving and loading circuits, as well as quitting the application.

The language menu allows choosing the user interface tongue (needs relaunching the application).

The help menu proposes an about dialog, as well as inline help pages.

Lastly, the edit menu allows toggling the logs dock on and off, clearing the contents of the main area, clearing the logs, and two experimental functionalities:
Add IOs tries to add as many inputs and outputs as are still needed to complete a circuit. (ie. if you add a NOT gate to the main area, then use this option, it will add one input and one output).
Batch rename is used to name a quantity of inputs or outputs at once. It could be used in an 8-bit adder as follows : add 8 inputs for the first number, select them all, click batch rename, and propose "A" - they will now be named A0 to A7, do the same for the second number, propose "B", and have them now named B0 to B7.

The last entry in the edit menu is "Settings", which allows the user to choose the clock speed, verbosity options and interface colors.
